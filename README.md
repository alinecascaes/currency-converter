# Convert - Conversor de Moedas

Aplicação web para conversão de moedas desenvolvida com HTML, CSS e JavaScript puro. Permite converter valores de Dólar Americano, Euro e Libra Esterlina para Real Brasileiro de forma rápida e intuitiva, exibindo a cotação utilizada e o valor convertido em uma interface moderna e responsiva.

## Funcionalidades

- Conversão de moedas estrangeiras para Real Brasileiro (BRL);
- Suporte para:
  - Dólar Americano (USD);
  - Euro (EUR);
  - Libra Esterlina (GBP);
- Validação do campo de entrada para permitir apenas números;
- Exibição da cotação utilizada na conversão;
- Resultado formatado em moeda brasileira;
- Tratamento de erros para valores inválidos;
- Interface moderna com foco em simplicidade e usabilidade;
- Layout responsivo e centralizado.

## Decisões Técnicas

- **HTML5 semântico** para estruturação da aplicação;
- **CSS3 puro** para estilização da interface;
- Uso de **Flexbox** para alinhamento e organização dos elementos;
- **JavaScript Vanilla** para manipulação do DOM e lógica de conversão;
- Utilização de `toLocaleString` para formatação monetária em Real Brasileiro;
- Separação simples entre estrutura (`HTML`), estilo (`CSS`) e comportamento (`JavaScript`);
- Controle visual de exibição do resultado através de manipulação de classes CSS.

## Tecnologias

- HTML5
- CSS3
- JavaScript
